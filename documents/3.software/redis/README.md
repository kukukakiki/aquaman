# Redis

## 下载&安装

```
$>tar xzf redis-3.2.5.tar.gz
$>cd redis-3.2.5/ 
$>make
```

## 启动

已"&"符号结尾,表示后台启动

```
$>cd redis-3.2.5/ 
$>src/redis-server &
```

## 客户端

### 连接redis服务

```
$>cd redis-3.2.5/ 
$>src/redis-cli
127.0.0.1:6379>
```

### 授权

当redis需要授权是，登陆客户端需要执行一下命令，才能执行后续的redis操作

```
127.0.0.1:6379>auth #your password#
ok
```

### 新增值

参数一set是redis关键字,第二个参数是key值,参数三是key对应的值

```
127.0.0.1:6379>set test test111
```


### 获取值

参数一get是redis关键字,第二个参数是key值,控制台输出key对应的值

```
127.0.0.1:6379>get test
"test111"
127.0.
0.1:6379>
```

### 删除值

```
127.0.0.1:6379>del test
(integer) 1
```

### 获取列表key

```
127.0.0.1:6379>keys *
1)"test"
2)"test1"
```
